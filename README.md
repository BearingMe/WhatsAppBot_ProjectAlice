<br>

### Installing

Clone this project

```bash
> git clone https://github.com/Coding-in-community/WhatsAppBot_ProjectAlice.git
> cd WhatsAppBot_ProjectAlice.git
```

Install dependencies:

```bash
> npm install
```

### Using

```bash
> node index.js
```

Once the bot is running you can scan the qr code with whatsapp
